# Nick's Dotfiles

## Installation

Clone this repo to `~/.dotfiles`

```bash
cd ~/.dotfiles
rake install
```

Also install GUI Applications

* GitHub
* VSCode
* Doppler for Secrets

## VSCode Settings

> Check out the vscode.json file in this repo

Also:

* Install Shell Command within VSCode to open files using `code .`
* Install VSCode Vim and ESLint
